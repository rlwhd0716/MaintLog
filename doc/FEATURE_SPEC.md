# 📄 FEATURE_SPEC.md

## 1. 공통 기능

- 앱 실행 및 초기 데이터 로드
- 하단 네비게이션 이동

## 2. 홈 (Dashboard)

- 업무 요약 정보 조회
- 최근 요청사항 표시
- 최근 장애 내역 표시
- 빠른 등록 기능

## 3. 요청사항 (Request)

### 기능
- 요청 등록
- 요청 목록 조회
- 요청 상세 조회
- 상태 변경
- 요청 수정

---

## 4. 작업 로그 (RequestLog)

- 로그 등록
- 로그 조회
- 로그 수정
- 로그 삭제

---

## 5. 장애 (Incident)

### 기능
- 장애 등록
- 장애 목록 조회
- 장애 상세 조회
- 상태 변경

## 6. 검색 (Search)

- 통합 검색 (요청 + 장애)
- 상태 필터
- 정렬 기능

## 7. 설정 (Settings)

- 사용자 설정
- 데이터 초기화
- 앱 정보 확인
